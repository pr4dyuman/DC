Design a premium, human-designed SaaS website for ZSEO.ai.

ZSEO.ai is an AI-first SEO operating system with 200+ SEO tools. The design must organize a large SEO tool inventory into clear product suites, workflows, and searchable tools. The result should feel like a mature product website and real software interface, not an AI-generated landing page.

Brand:
ZSEO.ai

Core idea:
ZSEO.ai is a command center for SEO work. Users can audit a website, find SEO issues, research keywords, monitor competitors, create content opportunities, track AI search visibility, manage reports, and connect external SEO data sources from one workspace.

Audience:
- SEO agencies
- In-house SEO teams
- Content teams
- Founders and small businesses
- Freelancers
- Technical SEO specialists
- Growth teams

Design personality:
Classic, calm, sharp, trustworthy, editorial, product-led, and highly organized.

The design should feel like it was created by an experienced SaaS product designer who understands SEO software. It should not feel like a generic AI website template.

Avoid these AI-generated website patterns:
- Purple/blue gradient hero backgrounds
- Glowing orbs, abstract blobs, mesh gradients, sparkles, neon halos
- Random floating 3D shapes
- Glassmorphism everywhere
- Huge rounded cards with repeated generic icons
- Overly large empty hero sections
- Fake futuristic AI decoration
- Decorative network lines or circuit graphics
- Generic "AI magic" sections
- Symmetrical card grids that feel automatically generated
- Dashboard mockups that look decorative instead of usable

Visual system:
- Background: white and warm off-white only
- Text: deep charcoal, near-black, and neutral gray
- Accent: one restrained color only, preferably refined green, cobalt, or teal
- Borders: crisp 1px neutral borders
- Radius: modest 6px to 8px
- Shadows: very subtle, used only for layered product surfaces
- Typography: serious SaaS/editorial typography with controlled scale
- Layout: strong grid, clean alignment, useful density
- UI style: practical dashboards, tables, command bars, filters, issue lists, score cards, and tool rows

AI design principle:
AI should appear as useful workflow intelligence, not as decoration. Show AI through:
- A global command/search bar
- Smart recommended actions
- Tool suggestions
- Prioritized issue lists
- AI-generated summaries
- Automation status
- "Ask ZSEO" assistant panel

Do not use AI as a visual gimmick.

Main navigation:
- Logo: ZSEO.ai
- Tools

- AI SEO

- Pricing

Right side:
- Sign in
- Start Free Audit
